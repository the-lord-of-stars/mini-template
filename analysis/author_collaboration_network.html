About This Visualization

This network visualization shows collaboration relationships between authors in visualization research papers. Each node represents an author, and each link represents a collaboration on a paper. Node size indicates the number of papers authored, node color indicates citation impact (red=high, green=low), and link thickness shows the number of collaborations between authors.

🔍 Zoom & Navigation Controls

  • Mouse wheel: Zoom in/out
  • Drag on empty space: Pan the view
  • Zoom buttons: Use +/− buttons or ⌂ to reset
  • Double-click: Zoom to fit all nodes

Interactions: Drag nodes to reposition them, hover over nodes/links for details, click nodes to highlight their connections, and use the filters above to explore different subsets of the data.
