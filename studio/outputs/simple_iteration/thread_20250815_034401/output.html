Iteration 1

Question: How do the citation counts (AminerCitationCount and CitationCount_CrossRef) of sensemaking papers vary across different years in the dataset?
ðŸ“Š Analysis Code (click to expand)
import pandas as pd
import matplotlib.pyplot as plt

# Load the dataset
df = pd.read_csv('outputs/simple_iteration/thread_20250815_034401/dataset_selected.csv')

# Filter papers related to sensemaking
df_sensemaking = df[df['Title'].str.contains('sensemaking', case=False, na=False)]

# Group by year and calculate average citation counts
yearly_citations = df_sensemaking.groupby('Year')[['AminerCitationCount', 'CitationCount_CrossRef']].mean()

# Plot the citation counts over the years
plt.figure(figsize=(10, 6))
plt.plot(yearly_citations.index, yearly_citations['AminerCitationCount'], label='Aminer Citation Count', marker='o')
plt.plot(yearly_citations.index, yearly_citations['CitationCount_CrossRef'], label='CrossRef Citation Count', marker='o')
plt.title('Average Citation Counts of Sensemaking Papers Over Years')
plt.xlabel('Year')
plt.ylabel('Average Citation Count')
plt.legend()
plt.grid(True)
plt.show()

# Print facts
print("### Begin of facts")
print(f"Years covered: {yearly_citations.index.min()} to {yearly_citations.index.max()}")
print(f"Average Aminer Citation Count: {yearly_citations['AminerCitationCount'].mean():.2f}")
print(f"Average CrossRef Citation Count: {yearly_citations['CitationCount_CrossRef'].mean():.2f}")
print("### End of facts")
